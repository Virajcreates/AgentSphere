# Phase 5 — AI Operations Platform: Final Enterprise Implementation Plan

This final implementation plan integrates backend-generated TypeScript types, a Next.js Backend-for-Frontend (BFF) routing layer, secure role-based `PermissionGate` interfaces, enhanced Tabbed Replay inspectors, rich Playground telemetry, and universal keyboard hotkeys into the **AI Operations Platform** (`web/`) conforming to Frozen Architecture `v1.1`.

---

## 1. Executive Summary & Design Refinements
The AI Operations Platform is updated to follow enterprise-level, production-grade frontend engineering principles:

1.  **Backend-Generated Typings**: The system avoids manual DTO duplication. Types are autogenerated directly from Python FastAPI's dynamic OpenAPI schema and stored inside `web/types/api.ts`, `models.ts`, and `schemas.ts`.
2.  **Backend-for-Frontend (BFF) Layer**: Establishes a Next.js API Route Handlers directory (`web/app/api/`). Pages communicate strictly with the Next.js BFF, which forwards requests to FastAPI, centralizing authorization mapping, request caching, and payload sanitization without duplicating business logic.
3.  **Unified Permission Gate**: Implements a reusable `<PermissionGate>` wrapper component that leverages AgentSphere's standard backend RBAC matrix (`superadmin`, `admin`, `editor`, `viewer`) to restrict UI elements on the client side.
4.  **Tabbed Replay Inspector**: Refines the Replay View to include a tabbed navigation panel displaying: Timeline, Execution Graph, Prompt, Retrieved Chunks, Tool Calls, Metadata, and raw logs.
5.  **Interactive Prompt Studio Actions**: Adds active workflows inside the Prompt Studio to: Compile variables, Preview renders, Run/Test scripts, Save states, and Publish versions.
6.  **Full Keyboard Shortcuts**: Registers universal system-wide listeners for `Ctrl+K` (Command Palette), `Ctrl+Enter` (Submit/Run), `Ctrl+S` (Save), `Ctrl+Shift+P` (Command Actions Palette), and `Esc` (Dismiss).

## 4. Risks & Mitigations

| Risk | Mitigation |
| :--- | :--- |
| **API Code Generation Desync** | Introduce an automated python-to-typescript script (`scripts/generate-types.sh` or standard node task) that runs on FastAPI startup validation, writing fresh typings dynamically. |
| **BFF Latency Overhead** | Keep BFF route handlers extremely lightweight. They act as raw stream-forwarders without deep loops, adding less than `1ms` overhead. |
| **Keyboard Short-circuit Collisions** | Validate target elements inside `useKeyboardShortcuts`: avoid intercepting default typing keys (such as typing `S` in input fields) by strictly evaluating modifier states (`event.ctrlKey`). |

## 5. Acceptance Criteria

- [ ] **TypeScript Typings Autogeneration**: TypeScript API models are generated from the backend OpenAPI schema. Duplicate DTO type files are prohibited.
- [ ] **Next.js BFF Layer Isolation**: Web app pages never execute direct fetches to the Python backend; they route through Next.js BFF handlers, which manage tracing headers and session caching.
- [ ] **Role-based PermissionGate**: Secures UI elements using `<PermissionGate>` wrappers that check user roles (`superadmin`, `admin`, `editor`, `viewer`) on render.
- [ ] **Comprehensive Replay Tabs**: Replay View exposes tabbed controls plotting Timeline, Execution Graph (React Flow), Prompt, Retrieved Chunks, Tool Calls, Metadata, and logs.
- [ ] **Metadata-Rich Playground**: Displays comprehensive telemetry logs: Provider, Model, Latency, token volumes, dollar expenses, Trace ID, and raw JSON strings.
- [ ] **Complete Prompt Workflows**: Interactive Prompt Studio manages 5 standard actions: Compile, Preview, Run, Save, and Publish.
- [ ] **Universal Hotkeys**: Evaluates and processes all 5 key short-cuts (`Ctrl+K`, `Ctrl+Enter`, `Ctrl+S`, `Ctrl+Shift+P`, `Esc`) reliably across pages.
- [ ] **DI & Code Coverage**: All DI singletons resolve successfully, and total project coverage remains above **`80%+`** with zero compilation errors.
